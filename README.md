# Обработка Excel-файлов с нормализацией регионов (LLM)

## Описание проекта

Решение автоматизирует обработку Excel-файлов с данными по регионам:

- Объединяет данные из нескольких листов
- Нормализует названия регионов с помощью LLM
- Сопоставляет регионы с ID из справочника
- Формирует итоговый файл и отчёт по качеству данных

---

## Структура проекта

### 1. `data_cleaner.py`

Скрипт для предварительной обработки исходного Excel-файла:

- Удаляет лишние строки
- Приводит структуру листов к единому виду
- Формирует файл `Cleaned Data.xlsx`

---

### 2. `main.py`

Основной скрипт:

- Читает очищенный Excel-файл
- Извлекает данные (регион, источник, сегмент)
- Отправляет данные в LLM (через Ollama)
- Получает нормализованный результат
- Сохраняет результат в `Result.xlsx`

---

### 3. `generate_report.py`

Скрипт генерации отчёта:

- Считает статистику
- Выделяет неизвестные регионы
- Находит аномалии
- Формирует файл `Report.xlsx`

---

## Входные файлы

Разместите в корне проекта:

- Исходный Excel-файл, например: `Start.xlsx`
- `Regions number.xlsm` - справочник регионов (название -> ID)
- `prompt.txt` - Промпт для нормализации данных
- `requirements.txt` - Зависимости проекта

После обработки появятся:

- `Cleaned Data.xlsx`
- `Result.xlsx`
- `Report.xlsx`

---

## Установка зависимостей

Убедитесь, что у вас установлен Python 3.10+.

Установите платформу Ollama: https://ollama.com/download/windows

В Olamma установите модель deepseek-r1:8b: через графический интерфейс или командную строку:
```bash
ollama pull deepseek-r1:8b
```
Установите зависимости:

```bash
pip install -r requirements.txt
```
---

## Запуск
### 1. Очистка исходного файла
```bash
python data_cleaner.py
```
Результат:
`Cleaned Data.xlsx`

### 2. Основная обработка через LLM

Перед запуском убедиться, что запущен Ollama:
```bash
ollama serve
```
Затем:
```bash
python main.py
```
Результат:

`Result.xlsx`

### 3. Генерация отчёта
```bash
python generate_report.py
```
Результат:

`Report.xlsx`
